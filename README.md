1. Présentation générale du site

Ce site est spécialisé dans l’apprentissage des mathématiques, destiné principalement aux élèves du collège au lycée, mais aussi aux étudiants souhaitant consolider leurs bases.

L’objectif du site est de rendre les mathématiques plus accessibles grâce à une approche combinant :

* des cours personnalisés en fonction de l'élève (en ligne) ;
* des corrections détaillées ;
* un suivi personnalisé des progrès.

Le site cherche à répondre à un problème fréquent chez les élèves : comprendre les mathématiques ne consiste pas seulement à apprendre des formules, mais à développer un raisonnement logique et une méthode de résolution.

2. Public visé

Le site s’adresse à plusieurs profils :

Élèves du collège :

Pour revoir les fondamentaux :

* calcul numérique ;
* fractions ;
* proportionnalité ;
* équations ;
* statistiques.

Élèves du lycée :

Avec des parcours adaptés aux spécialités :

* seconde générale ;
* première spécialité mathématiques ;
* terminale spécialité mathématiques.


Les thèmes proposés :

* fonctions ;
* suites ;
* probabilités ;
* dérivation ;
* géométrie dans l’espace ;

Étudiants :

Pour reprendre les bases nécessaires :

* analyse ;
* algèbre ;
* logique ;

4. Design du site

Le design doit être :

* clair ;
* moderne ;
* rassurant.

Couleurs possibles :

* bleu : confiance ;
* blanc : simplicité ;
* vert : progression.


5. Objectif final du site

Le but de ce site n’est pas seulement de donner des réponses.

Il veut apprendre aux élèves à :

* comprendre ;
* raisonner ;
* résoudre ;
* devenir autonomes.
